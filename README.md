# RoboClicker

RoboClicker — браузерная clicker-игра на HTML, CSS и JavaScript. Игрок зарабатывает золото кликами, улучшает силу клика, открывает скины, получает достижения и развивает прогресс.

## Возможности

- клики, улучшения и автоматический доход;
- уровни, достижения и скины;
- настройки языка, звука, яркости и темы;
- звуковые эффекты;
- локальный режим и интеграция с Яндекс Игры SDK;
- сохранение прогресса и обработка пауз/рекламы.

## Файлы

```text
index.html   интерфейс игры
style.css    оформление и адаптивность
script.js    логика игры и интеграция SDK
*.wav        звуковые эффекты
```

## Запуск

Откройте `index.html` через локальный сервер. Для простой проверки подойдет Live Server в VS Code.

## Публикация

Для публикации на Яндекс Играх в архив должны попасть `index.html`, `style.css`, `script.js` и звуковые файлы.
